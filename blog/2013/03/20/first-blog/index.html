izhong个人博客建站记录 时间一分一秒地过去，总想留下点什么。于是买了个域名，建了个博客。 一、购买域名
我使用的是godaddy（http://www.godaddy.com/），

点击左上角的登陆按钮，（没有帐户的先注册）购买域名。购买成功后可以看见管理页面，点击LUNCH按钮进入域名管理

找到自己购买后的域名点击进入后，首先设置Nameservers，这里我是用的是Dnspod来管理自己的域名，Dnspod的Dns是F1G1NS1.DNSPOD.NET ，F1G1NS2.DNSPOD.NET 。这样当请求域名解析的时候会自动将解析请求转到Dnspod的域名服务器。 二、进入Dnspod的网站设置域名解析
网址是www.dnspod.cn,登陆后添加自己的域名，设置解析记录，添加一个新的解析，这里主机记录为@表示是主域名，系统记录A，线路类型默认，记录值204.232.175.78，这是github的ip地址。当然也可以使用三级域名，添加CNAME记录。github绑定域名方法，在master分支下建立CNAME文件并输入自己的域名。 三、进入github添加一个Repository
添加的Repository的名字是xxx.github.com其中xxx是申请github的用户名，我添加的Repository是jimmyzhong.github.com，这样当访问jimmyzhong.github.com时将直接读取该Repository下面master分支的内容。 四、安装Octopress
到rubyinstall.org下载ruby的安装包，我下载的是RubyInstaller 1.9.3,一直点击下一步，最后一步将Path加入到系统环境变量勾选。并下载DevKit-tdm-32-4.5.2-20111229-1559-sfx.exe安装。我安装在D：/Devkit 打开cmd运行： d: cd Devkit ruby dk.rb init ruby dk.rb install 改变环境变量： gem sources -a http://ruby.taobao.org/ gem sources -r http://rubygems.org/ 改变环境变量的代码： set LANG=zh_CN.UTF-8 set LC_ALL=zh_CN.UTF-8 下载Octopress源码。使用git下载地址为git://github.com/imathis/octopress.git 进入Octopress目录执行 gem install bundler bundle install rake setup_github_pages rake install rake generate rake preview rake deploy 保存源码
